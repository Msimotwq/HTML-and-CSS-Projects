# HTML and CSS Projects

This repository contains projects and exercises completed during the HTML and CSS course.

## Projects

### Bootstrap Cinema Website

A responsive cinema-themed website built using Bootstrap 4. The project demonstrates navigation bars, cards, grids, forms, badges, responsive layouts and Bootstrap components.

### Challenge Projects

A collection of HTML and CSS exercises covering:

* Image styling and positioning
* CSS filters and border radius
* Footer creation and styling
* Tables and table formatting
* Text overlays on images
* Fixed navigation bars
* Two-column layouts
* Hover effects
* Image overlay effects
* HTML5 video integration
* Video text overlays
* Media queries and responsive design

## Skills Demonstrated

* HTML5
* CSS3
* Responsive Design
* Bootstrap 4
* Navigation Menus
* Forms
* Tables
* Images and Video
* Hover Effects
* Media Queries
* Git and GitHub
